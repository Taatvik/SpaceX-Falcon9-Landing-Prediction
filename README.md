🚀 SpaceX Falcon 9 First Stage Landing Prediction
Project Overview

This project was completed as part of the IBM Data Science Professional Certificate Capstone. The objective was to predict whether the first stage of a SpaceX Falcon 9 rocket would successfully land after launch. Accurate landing predictions are important because reusability significantly reduces launch costs and is a key factor in SpaceX's business model.

Using historical launch data, exploratory data analysis, data visualization, and machine learning techniques, a predictive model was developed to estimate the likelihood of successful first-stage recovery.

Problem Statement

SpaceX advertises Falcon 9 launches at a significantly lower cost than traditional launch providers due to its ability to reuse rocket boosters.


Project Workflow
1. Data Collection
Collected launch data from the SpaceX API.
Gathered additional launch information through web scraping.
2. Data Wrangling
Cleaned and transformed raw launch data.
Handled missing values and inconsistent records.
Selected relevant launch features.
3. Exploratory Data Analysis (EDA)
Investigated relationships between launch variables and landing outcomes.
Analyzed launch sites, payload mass, orbit types, and flight history.
Created visualizations to identify important patterns.
4. Interactive Visual Analytics
Built interactive dashboards and visualizations.
Explored landing success rates across different launch conditions.
5. Machine Learning
Trained classification models to predict landing success.
Compared model performance using standard evaluation metrics.
Identified the most influential factors affecting successful landings.
6. Results & Insights
Evaluated predictive performance.
Derived insights regarding launch characteristics that contribute to successful booster recovery.

Results

The project successfully developed a machine learning pipeline capable of predicting Falcon 9 first-stage landing outcomes using historical launch data. The analysis highlighted the impact of payload mass, launch site, orbit type, and previous flight history on landing success.
